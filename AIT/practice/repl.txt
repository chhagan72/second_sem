*REPL in Node js
The REPL feature of node is very useful in experimenting with Node.js  codces and  to debug javascript codes.

REPL(Read, Eval, Print, Loop).

1. Read - Read user's input, parses the input into javaSript data-structure, and store in memory.

2. Eval - Takes and Evaluates the data structure.

3. Print - Prints the result.

4. Loop - Loops the above command until the user presses ctl-c twice.
